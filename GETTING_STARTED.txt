🚀 TO RUN YOUR WEBSITE - FOLLOW THESE 3 STEPS:

STEP 1: Set your API Key
────────────────────────
export GROQ_API_KEY="gsk_your_key_here"

Get free key: https://console.groq.com


STEP 2: Start Backend (Terminal 1)
──────────────────────────────────
cd backend
python main.py

Expected output:
  ✅ Semantic model loaded
  ✅ Qdrant vector database initialized
  ✅ Groq chat model initialized
  INFO:     Uvicorn running on http://0.0.0.0:8000


STEP 3: Start Frontend (Terminal 2)
───────────────────────────────────
cd frontend
python -m http.server 3000

Expected output:
  Serving HTTP on 0.0.0.0 port 3000


THEN OPEN IN BROWSER:
→ http://localhost:3000

🔧 TROUBLESHOOTING:

❌ "Connection refused"
  → Make sure backend is running: cd backend && python main.py

❌ "API Key not found"
  → export GROQ_API_KEY="your_key_here"

❌ "Module not found"
  → pip install -r backend/requirements.txt

❌ "Port already in use"
  → Backend: change port in backend/main.py
  → Frontend: python -m http.server 3001
